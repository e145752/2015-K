現在、各プロジェクト(フォルダ)には、以下のものが置いてあります。
 Animation	- 各種アニメーション系(現在テストが置いてあります)
 Editor		- エディタ系を司るためのコード
 Fonts		- フォント(現在は適当に拾ってきたもの)
 Materials	- Unity内で作成した物体(現在は空)
 Prefabs	- 複数のアイテムをひとまとめにしたもの(現在テストが置いてあります)
 Scenes		- 現在の作成状態を保存しています。とりあえずこれを実行してみてください。
 Scripts	- 作成したメイン機能のソースコード
 Sounds		- ゲーム内SE,BGM(現在は空)
 Sprites	- 各種素材(主に画像)が置いてあります
 StandardAssets	- バーチャルパッドのソースコード。割と手を加えてます。
 Textures	- テクスチャ系(現在は空)

すでに配置してあるオブジェクトに関しては、サイズ・位置・判定など、微調整済みです。
ヒエラルキーに関しては、日々大きく変更が加わるため、聞かれればその都度説明をしたいです。
